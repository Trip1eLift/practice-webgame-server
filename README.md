# websocket-concurrent-server
Use python as the backend concurrent server


## Game Server  5001
To setup python virtual environment
```console
cd server
python3 -m venv ./venv
```

To activate
```console
where.exe python
./venv/Scripts/activate     (Windows)
source ./venv/bin/activate  (Mac)
where.exe python
```

Operations in venv
```console
pip list
pip install -r requirements.txt
```

```console
deactivate
```

## React Client 3000
